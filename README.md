# DataAnalysis in Excel using Dashboard
This project is an Excel-based Data Analysis Dashboard created to analyze and visualize business data using Excel tools. It demonstrates my ability to clean data, analyze trends, and present insights in a clear, interactive dashboard format.
## Dataset used - 
< a href = "https://github.com/GauriRandhir/DataAnalysis-Dashboard/blob/main/Book1.xlsx"> Dataset </a>


## Tools & Features Used
Microsoft Excel
Pivot Tables
Pivot Charts
Excel Formulas (SUM, COUNT, IF, etc.)
Data Cleaning & Formatting
Slicers for interactivity
Dashboard layout & design

## Key Insights Provided
Sales / performance summary
Category-wise and region-wise analysis
Trend analysis using charts
Interactive filtering using slicers
Quick decision-making visuals

## Objective
The goal of this project is to:
Practice data analysis using Excel
Build a professional dashboard
Showcase skills relevant to Data Analyst / Business Analyst roles

## Skills Demonstrated
Data Analysis
Data Visualization
Business Insights
Excel Dashboard Design
Analytical Thinking
